# 🚀 DevBiel Mini Projects

Bem-vindo ao repositório de projetos rápidos do **DevBiel** 👨‍💻🔥

Aqui você vai encontrar **mini projetos em HTML, CSS e JavaScript**, feitos para:

* Aprender na prática
* Usar em seus próprios projetos
* Servir de inspiração
* Melhorar seu portfólio

---

## 📌 Objetivo

Criar projetos simples, bonitos e reutilizáveis para ajudar devs iniciantes e intermediários a evoluírem mais rápido 🚀

---

## 🧠 O que você vai encontrar aqui?

✔️ Componentes de UI
✔️ Animações em CSS
✔️ Projetos interativos com JavaScript
✔️ Landing pages simples
✔️ Ideias rápidas para portfólio

---

## 📂 Estrutura do projeto

Cada projeto fica em sua própria pasta:

```
/nome-do-projeto
 ├── index.html
 ├── style.css
 └── script.js
```

---

## 🚀 Como usar

1. Clone o repositório:

```
git clone https://github.com/seu-usuario/devbiel-mini-projects.git
```

2. Acesse a pasta do projeto desejado

3. Abra o arquivo `index.html` no navegador

Pronto 🎉
